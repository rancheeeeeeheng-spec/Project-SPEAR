## Versioning

Project S.P.E.A.R. uses Semantic Versioning.

| Version   | Description                                       |
| --------- | ------------------------------------------------- |
| **MAJOR** | Significant architectural or functional changes   |
| **MINOR** | New features and enhancements                     |
| **PATCH** | Bug fixes, optimisations, and maintenance updates |

Example:

* **1.0.0** – Initial public release
* **1.1.0** – New features
* **1.1.1** – Bug fixes
* **2.0.0** – Major platform enhancement
